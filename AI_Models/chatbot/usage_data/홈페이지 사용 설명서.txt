[전체 페이지 공통 사항]
- 우측 상단에 "공지사항"과 "알림" 버튼이 있음.
- "공지사항" 클릭 → "공지사항 페이지"로 이동. 단, 로그인이 된 상태에서만 이동. 미로그인 시 로그인 페이지로 이동
- "알림" 클릭 → "알림 페이지"로 이동. 단, 로그인이 된 상태에서만 이동. 미로그인 시 로그인 페이지로 이동
- 페이지 좌측 상단에는 "사이트 로고"(누르면 "메인 페이지" 이동)가 있다.
- "메인페이지"를 제외한 페이지 좌측의 "MENU"바로 밑에는 "대시보드", "공고찾기", "장바구니", "커뮤니티" 버튼이 있다.
- "대시보드" 클릭 → "대시보드 페이지"로 이동. 단, 로그인이 된 상태에서만 이동. 미로그인 시 로그인 페이지로 이동
- "공고찾기" 클릭 → "공고찾기 페이지"로 이동.
- "장바구니" 클릭 → "장바구니 페이지"로 이동. 단, 로그인이 된 상태에서만 이동. 미로그인 시 로그인 페이지로 이동
- "커뮤니티" 클릭 → "커뮤니티 페이지"로 이동.
- 페이지 좌측 하단에는 "마이페이지"와 "로그아웃" 버튼이 있다. 단, 미로그인시 "로그인" 버튼만 있다.
- "마이페이지" 클릭 → "마이페이지"로 이동.
- "로그아웃" 클릭 → 로그아웃 상태가 되어 "메인페이지"로 이동.
- 화면 우측 하단에는 "AI 도우미" 버튼이 있다. 
- 페이지 우측 하단 "AI 도우미" 버튼 바로밑에는 "이용약관", "개인정보처리방침", "고객지원" 버튼이 있다.
- "AI 도우미" 클릭 → "챗봇 모달" 열림
- "이용약관" 클릭 → "서비스 이용약관 페이지"로 이동
- "개인정보처리방침" 클릭 → "개인정보처리방침 페이지"로 이동
- "고객지원" 클릭 → "고객지원 페이지"로 이동

[커뮤니티 페이지]
- 커뮤니티에 등록된 글의 목록과 정보를 확인할 수 있는 페이지
- 페이지 상단에서 글 유형(전체, 질문, 정보, 후기, 토론) 별로 필터링을 거쳐 게시글 목록 확인이 가능하다.
- 검색창을 통해 커뮤니티 글을 검색할 수 있다.
- "검색하기" 버튼 옆에 "최신순", "인기순", "조회순", "댓글순" 별로 게시글 목록을 조회할 수 있다.
- 검색창 아래에 인기글을 확인할 수 있다. 1위부터 3위까지 보여준다. 조회수와 조회 날짜가 인기글 순위에 영향을 준다.
- 게시글 목록에서 글의 유형, 글의 제목, 작성자, 작성일, 조회수, 댓글수, 좋아요 수에 대한 정보를 간략히 확인할 수 있다.
- 작성자에 작성자의 이름과 등급이 표시된다. 질문 글에 댓글 작성 후 채택이 5회되면 등급이 한 단계 상승한다. 등급의 종류는 곡괭이, 굴삭기, 지게차, 불도저, 포크레인 순으로 높다.
- 게시글을 클릭하면 게시글에 대한 상세 내용을 확인할 수 있는 "게시글 페이지"로 이동할 수 있다.
- 현재 로그인이 되어있는 상태라면 페이지 우측 상단의 "글쓰기" 버튼을 눌러 "새 글 작성 페이지"로 이동하여 커뮤니티에 게시글을 작성할 수 있다.
- 미로그인 상태이면 커뮤니티 페이지에서 글쓰기를 이용할 수 없다.
- 미로그인 상태이면 우측 상단은 "글쓰기" 버튼은 "로그인 후 글쓰기" 버튼으로 전환되며, 그것을 클릭하면 "로그인 페이지"로 이동한다.

[게시글 페이지]
- 커뮤니티에 게시된 글의 내용을 확인할 수 있는 페이지다.
- 이 페이지에서는 글의 유형, 글의 제목, 작성자, 작성자의 등급, 작성일, 조회수, 좋아요 수에 대한 정보를 글 본문 영역 상단에서 확인할 수 있다.
- 이 페이지의 하단에는 댓글을 작성할 수 있다. 
- 댓글 작성 영역에는 현재 작성된 댓글의 갯수를 확인할 수 있다.
- "댓글 입력칸(로그인시 "댓글을 입력하세요..." , 미로그인시 "로그인 후 댓글을 작성할 수 있습니다" 라고 적힌 텍스트 박스)"에서 글을 입력한 다음 "댓글 작성" 버튼을 클릭해 댓글을 등록할 수 있다. 단, 로그인 시에만 가능하며 미 로그인시 "댓글 작성" 버튼은 비활성화 된다.
- 등록한 댓글은 페이지 하단에 표시된다.
- 로그인 되어있는 사용자가 현재 게시글의 작성자라면, 글 본문 영역 우측 상단 "삭제" 버튼을 클릭해 커뮤니티에서 글을 삭제할 수 있다.
- 로그인 되어있는 사용자가 현재 게시글의 작성자라면, 글 본문 영역 우측 상단 "수정" 버튼을 눌러 게시글의 내용을 수정할 수 있다.
- 로그인시 글 본문 영역의 "좋아요" 버튼을 누르면 좋아요 수가 1만큼 오르고, "좋아요" 버튼은 "좋아요 취소" 버튼이 되며 "좋아요 취소" 버튼을 누르면 좋아요 수가 1만큼 줄어들고 다시 "좋아요" 버튼 상태로 돌아간다.
- 미로그인시 "좋아요" 버튼이 "로그인 후 좋아요" 로 바뀌고 좋아요를 할 수 없다.

[계정 찾기 페이지]
- 가입 시 등록한 정보로 계정(이메일)을 찾을 수 있는 페이지다.
- 가입 시 등록되어 있는 "이름"과 "생년월일"을 입력하여 "다음" 버튼을 누르면 가입 시 설정한 질문을 확인 할 수 있으며, 이에 맞는 답변을 작성 후 "계정 확인"을 누르면 계정(이메일)을 알려준다. 혹은 "다시 입력하기" 버튼을 눌러 "이름"과 "생년월일"을 다시 입력할 수 있다. 
- 계정확인을 누른 후 "로그인하러 가기" 버튼을 통해 로그인 페이지로 이동할 수 있고, "다시 찾기" 버튼을 눌러 "이름"과 "생년월일"을 다시 입력할 수 있다.  

[공고찾기 페이지]
- 현재 사이트에 등록되어 있는 공고들을 확인할 수 있는 페이지다.
- 이 페이지에서는 각 공고의 마감일, 공고명, 발주기간, 예산, 상태에 대한 간략한 정보를 확인할 수 있다. 
- "액션"의 "담기" 버튼을 클릭하여, 해당하는 공고를 자신의 "장바구니"에 등록시킬 수 있다.
- 등록되어 있는 공고를 클릭하거나, "상세" 버튼을 클릭하면 해당 공고에 대한 상세정보를 보여주는 페이지로 이동한다.
- 키워드(공고명/기관/예산/마감)를 입력하여 이에 해당하는 공고들만 표시할 수 있다.
- 페이지 우측에서 "기관명", "마감일", "공고 개수" 별로 필터를 사용해 공고 목록을 확인할 수 있다.
- "새로고침" 버튼을 클릭하면 공고 목록을 최신화 할 수 있다.
- "필터 초기화" 버튼을 클릭하면 입력한 키워드를 초기화 한다. 
- 페이지 하단에 공고 목록들의 페이지 번호가 나온다. 직접 숫자를 클릭해 공고 목록 페이지를 이동하거나 Previous를 클릭해 페이지 번호를 -1 할 수 있고, Next를 클릭해 페이지 번호를 +1 할 수 있다.

[공고 상세 페이지]

[공지사항 페이지]
- 지금까지 등록된 공시사항을 확인할 수 있는 페이지다.
- 이 페이지에서는 공지사항 이름, 공지사항 상태(태그), 작성일, 작성자, 첨부파일 개수를 간략히 확인할 수 있다.
- 검색창을 입력해 찾고싶은 공지사항을 찾을 수 있다.
- 검색창 밑의 버튼들을 사용해 태그(서비스, 업데이트, 점검, 정책)별로 공지사항을 검색할 수 있다.
- 공지사항 우측의 "상세" 버튼을 클릭하면 그 공지사항의 상세내용을 확인할 수 있다.
 
[대시보드 페이지]
- 이번 달의 신규공고, 장바구니 안에 등록된 공고(관심 공고) 수, 마감이 3일 이내에 있는 공고 수, 
장바구니 안에 등록된 공고(관심공고)의 예산 금액의 총 합계, 최근 6개월간의 월별 공고 추이, 현재 진행 중인 공고의 지역별 분포를 확인할 수 있는 페이지다.
- 신규 공고, 관심 공고, 마감 임박, 총 예상액은 페이지 상단에 숫자로 확인할 수 있다.
- "월별 공고 추이"는 월별로 등록된 공고 수를 선 그래프 형태로 보여준다.
- "지역별 분포"는 지역별로 등록된 공고 수를 막대 그래프 형태로 보여준다.

[로그인 페이지]
- 웹사이트에 로그인을 할 수 있는 페이지이다.
- 이 페이지는 비로그인 상태에서 "로그인이 필요한 기능"(장바구니, 대시보드, 글쓰기, 알림 등)에 진입하려 하면 나타나는 페이지다.
- 이메일과 비밀번호를 입력하여 로그인 할 수 있다.
- 자신의 계정이 없다면 하단의 "회원가입"을 눌러서 "회원가입 페이지"로 이동할 수 있다.
- 자신의 계정을 모를 때 "계정 찾기"클릭해 눌러 "계정 찾기 페이지"로 이동할 수 있다.
- 자신의 비밀번호를 모를 때 "비밀번호 찾기"를 클릭해 "비밀번호 찾기 페이지"로 이동할 수 있다.

[마이페이지]
- 현재 로그인한 사용자의 계정 정보 및 설정 제공하는 페이지다.
- 이 페이지 상단에는 이용자 이름, 이용자의 이메일, 계정 상태를 확인할 수 있다.
- 이 페이지에서 "계정 정보", "회사 정보", "알림 설정", "구독 관리" 버튼들을 클릭해 보여주는 정보를 전환할 수 있다.
- "계정 정보"에서는 현재 이용자 계정의 이름, 이메일 정보를 확인할 수 있다.
- "계정 정보"의 "비밀번호 변경"에서는, 현재 비밀번호와 새 비밀번호를 입력하고, 비밀번호 확인에 새 비밀번호와 동일하게 입력한 후 "변경사항 저장" 버튼을 누르면 현재 계정의 비밀번호가 "새로 설정할 비밀번호"로 갱신된다.
- "계정 정보"의 페이지 하단에 "화면 설정" 이 있다. 이 중 "야간 모드 (다크 모드)" 영역의 우측 스위치를 클릭해 테마들을 전반적으로 어둡게 바꾸어 사용 할 수 있으며, 다시 스위치를 클릭하면 기존 테마로 쓸 수 있다.
- "회사 정보"에는 회사명과 직위를 저장할 수 있다. 회사명과 직위를 입력해 회사 정보를 저장할 수 있다. 저장된 상태에서 새로 회사명과 직위를 입력하여 저장을 하면 회사 정보를 업데이트 할 수 있다.
- "알림 설정"에서는 알림 설정을 제공한다.
- "구독 관리"에서는 구독 관리를 제공한다.

[메인페이지]
- 공고 검색, 빠른 이동 제공, 추천 공고 확인, 로그인 상태를 확인하는 기능의 제공하는 페이지다.
- 페이지 상단에 공고를 키워드 (공고명, 기관, 예산, 마감)로 검색 할 수 있다.
- 페이지 중앙에 "바로가기"를 통해 사이트의 주요 기능(대시보드, 공고 찾기, 장바구니, 커뮤니티)에 빠르게 접근할 수 있다.
- "장바구니"에서는 현재 로그인한 계정에서 장바구니로 넣은 공고의 수를 파악할 수 있다. "장바구니" 영역을 클릭하면 "장바구니 페이지"로 이동한다. 미로그인시 "로그인 페이지"로 이동 후  로그인시 "장바구니 페이지"로 이동한다.
- "대시보드" 영역을 클릭하면 "대시보드 페이지"로 이동한다. 미로그인시 "로그인 페이지"로 이동 후  로그인시 "대시보드 페이지"로 이동한다.
- 페이지 하단 "오늘의 추천"에는 오늘 등록된 신규 공고의 수와 마감일까지 3일 이내의 기간이 남은 공고의 수를 파악할 수 있다.
- "오늘의 추천"에서 "신규 공고" 영역을 누르면이 "대시보드 페이지"로 이동해 "신규 공고"를 보여준다. 미로그인 시에는 "로그인 페이지"로 이동 후 로그인시 이동해 "대시보드 페이지"로 이동해 "신규 공고"를 보여준다.
- "오늘의 추천"에서 "마감 임박" 영역을 누르면 "대시보드 페이지로 이동해" "마감 임박 공고"를 보여준다. 미로그인 시에는 "로그인 페이지"로 이동 후 로그인시 "대시보드 페이지로 이동해" "마감 임박 공고"를 보여준다.
- 페이지 우측에 "로그인" 상태를 확인 할 수 있다. 미로그인시 이메일, 비밀번호를 입력해 "로그인" 버튼을 눌러 로그인을 할 수 있고, "회원가입 버튼"을 통해 "회원가입 페이지"로 이동, "계정 찾기" 버튼을 통해 "계정 찾기 페이지"로 이동, "비밀번호 찾기" 버튼을 눌러 "비밀번호 찾기 페이지"로 이동할 수 있다. 로그인시 "프로필 수정" 버튼을 눌러 "마이페이지"로 이동할 수 있고, "로그아웃"을 통해 로그아웃 할 수 있다.

[비밀번호 찾기 페이지]
- 임시 비밀번호를 이메일로 전송하기 전 본인 확인을 수행하는 페이지다.
- 이름, 이메일, 생년월일을 입력하고 "다음" 버튼을 누르면 가입 시 설정한 질문이 나오며 이에 대한 답변을 입력할 수 있다. 맞는 답변 입력 후 "임시 비밀번호 전송"을 누르면 임시 비밀번호를 메일로 받을 수 있다. "다시 입력하기" 버튼을 누르면 이름, 이메일, 생년월일을 다시 입력 할 수 있다. 
- "로그인" 버튼을 눌러 "로그인 페이지"로 이동 할 수 있고, "계정 찾기" 버튼을 눌러 "계정 찾기"로 이동할 수 있다.

[새 글 작성 페이지]
- 커뮤니티에 새로운 게시글을 작성할 수 있는 페이지다.
- "카테고리"의 버튼을 클릭하여 글의 "유형"을 설정할 수 있다.
- 제목과 글의 내용은 이용자가 직접 입력한다.
- "첨부파일"을 통해 이용자가 직접 파일을 업로드할 수 있다.
- "등록" 버튼을 클릭하면 사용자가 입력한 내용대로 커뮤니티에 게시글이 등록된다. (제목과 내용에서 한 곳이라도 입력이 되어있지 않으면 등록 불가)
- "취소" 버튼을 클릭하면 커뮤니티 페이지로 돌아간다.

[알림 페이지]
- 자신 계정의 장바구니(관심 공고)에 담긴 공고에서 변동사항이 발생했을 때 이에 대한 정보를 확인할 수 있는 페이지다.
- 공고의 현재 상태, 상태의 상세 내용, 상태 갱신 시간을 확인할 수 있다. 
- 마감 임박 : 공고의 마감일에 도달하기 3일 이내의 공고에 대한 알림
- 정정 공고 : 공고에 정정사항이 발생했을 경우의 알림
- 재공고 : 관심 공고 중에서 재공고된 공고의 수에 대한 알림
- 전체 알림 표시, 긴급 알림 표시로 전환할 수 있다.
- "설정"에서는 "마감 임박 알림"과 "정정/재공고 알림"을 동작시킬지 설정할 수 있다.
- "모두 읽음"을 눌러 알림을 전부 읽음 처리할 수 있다.

[장바구니 페이지]
- 현재 자신 계정의 장바구니에 등록된 관심 공고에 대한 정보를 확인 할 수 있는 페이지다.
- 페이지 상단에서 관심, 검토중, 참여결정, 서류준비, 제출완료, 낙찰, 탈락 상태 별 공고 개수가 표시된다.
- "장바구니 공고 목록"에서는 공고명, 마감일까지 남은 기간, 발주기관명, 예산, 마감일, 관심 공고 상태에 관한 정보를 간략히 확인할 수 있다.
- "마감 빠른순", "마감 늦은순", "제목순"으로 관심 공고 목록을 정렬할 수 있다.
- 공고를 직접 클릭하여 해당 공고의 상세 내용을 확인할 수 있다.
- 공고 우측에서 "태그 설정"과 "관심 공고 삭제"를 실행할 수 있다.
- 태그 설정 : 관심 공고의 현재 상태(관심, 검토중, 참여결정, 서류준비, 제출완료, 낙찰, 탈락)를 설정
- 휴지통 모양 버튼 : 클릭하면 관심 공고를 장바구니에서 삭제한다.

[챗봇 모달]
- 조건별 공고 검색, 사이트 사용 설명을 제공하는 챗봇 기능을 구현한 모달이다.
- 챗봇을 통해 알고싶은 내용을 입력하고 "전송" 버튼을 클릭하면, 챗봇이 질문에 맞는 답변을 생성한다.
- 모달의 우측의 "빠른 질문"에서 원하는 내용의 항목을 클릭하여 빠르게 질문을 입력하고 그에 맞는 답변을 출력하도록 할 수 있다.

[회원가입 페이지]
- 회원가입을 위한 정보를 입력하는 페이지다.
- 이름, 이메일, 닉네임, 생년월일을 입력한다.
- "비밀번호"는 아래의 "비밀번호 규칙" 체크리스트를 따라서 작성해야 된다.
- "비밀번호 규칙" 체크리스트에서 하나라도 만족하지 않을 시 "가입하기" 버튼은 활성화 되지 않는다.
- "비밀번호"를 체크리스트에 맞게 잘 작성했다면, "비밀번호 확인"에서 동일한 비밀번호로 입력한다.
- "계정 찾기 질문"을 클릭하여 내가 사용하고 싶은 질문을 선택한다.
- "계정 찾기 답변"에서 질문에 사용할 답변을 입력한다.
- "약관 동의"의 체크 박스를 클릭한다. (필수적으로 해당하는 항목 체크는 필수)
- 모든 정보를 입력했다면 "가입하기" 버튼이 활성화 되었을 것이고, 그것을 클릭하면 회원가입 절차는 완료된다.
