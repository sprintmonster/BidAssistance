    -로그인 상태: "공지사항 페이지"로 이동한다.
    -미로그인 상태: "로그인 페이지"로 이동한다.
- 알림 버튼 클릭 시:
    -로그인 상태: "알림 페이지"로 이동한다.
    -미로그인 상태: "로그인 페이지"로 이동한다.

[모든 페이지 공통 - 좌측 영역]
- 모든 페이지 좌측 상단에는 사이트 로고가 있다.
- 사이트 로고를 클릭 시 "메인 페이지"로 이동한다.
- "메인페이지", "회원가입 페이지", "로그인 페이지", "계정 찾기 페이지", "비밀번호 찾기 페이지"를 제외한 모든 페이지의 좌측 메뉴 바에는 다음 버튼이 있다.
    -대시보드
    -공고찾기
    -장바구니
    -커뮤니티

[모든 페이지 공통 - 좌측 메뉴 바 동작]
- 대시보드 클릭 시:
    -로그인 상태: "대시보드 페이지"로 이동한다.
    -미로그인 상태: "로그인 페이지"로 이동한다.
- 공고찾기 클릭 시:
    -"공고찾기 페이지"로 이동한다.
- 장바구니 클릭 시:
    -로그인 상태: "장바구니 페이지"로 이동한다.
    -미로그인 상태: "로그인 페이지"로 이동한다.
- 커뮤니티 클릭 시:
    -"커뮤니티 페이지"로 이동한다.

[모든 페이지 공통 - 좌측 메뉴 바 하단 영역]
- 좌측 메뉴 바가 있는 페이지 좌측 하단에는 다음 버튼이 있다.
    -로그인 상태: 마이페이지, 로그아웃
    -미로그인 상태: 로그인
- "마이페이지" 클릭 → "마이페이지"로 이동.
- 로그아웃 클릭 시 로그아웃 처리 후 "메인 페이지"로 이동한다.
- 페이지 최하단에는 다음 링크가 있으며 클릭 시 화면에 해당 내용을 보여준다.
    - 이용약관
    - 개인정보처리방침
    - 고객지원

[모든 페이지 공통 - AI 도우미]
- "회원가입 페이지", "로그인 페이지", "계정 찾기 페이지", "비밀번호 찾기 페이지"를 제외한 모든 페이지의 화면 우측 하단에 AI 도우미 버튼이 있다.
- AI 도우미 클릭 시 "챗봇 모달"이 열린다.

[커뮤니티 상세 페이지 - 개요]
- 커뮤니티에 등록된 글의 목록과 정보를 확인할 수 있는 페이지다.
- 게시글을 클릭하면 해당 게시글의 "게시글 상세 페이지"로 이동한다.

[커뮤니티 상세 페이지 - 게시글 필터]
- "커뮤니티 페이지" 상단에서 글 유형(전체, 질문, 정보, 후기, 토론) 별로 필터링할 수 있다.
- 검색창을 통해 커뮤니티 글을 검색할 수 있다.
- 검색하기 버튼 옆에 최신순, 인기순, 조회순, 댓글순으로 게시글 정렬을 할 수 있다.

[커뮤니티 상세 페이지 - 인기글]
- 검색창 아래에서 인기글을 1~3위를 확인할 수 있다.
- 조회수와 조회 날짜가 인기글 순위에 영향을 준다.

[커뮤니티 상세 페이지 - 게시글 목록 정보]
- 게시글 목록 다음 정보를 확인할 수 있다.
    - 글 유형, 제목, 작성자, 작성일, 조회수, 댓글수, 좋아요 수
- 작성자에 작성자의 이름과 등급이 표시된다. 

[커뮤니티 상세 페이지 - 작성자 등급]
- 질문 글에 댓글 작성 후 채택이 5회 누적될 때마다 등급이 상승한다. 
- 등급 순서는 곡괭이 → 굴삭기 → 지게차 → 불도저 → 포크레인 순이다.

[커뮤니티 상세 페이지 - 글쓰기 버튼]
- 로그인 상태:
    - "커뮤니티 페이지"의 우측 상단에 글쓰기 버튼을 눌러 "새 글 작성 페이지"로 이동한다.
- 미로그인 상태:
    - "로그인 후 글쓰기" 버튼으로 표시된다.
    - 클릭 시 "로그인 페이지"로 이동한다.

[게시글 상세 페이지 - 개요]
- "커뮤니티 페이지"에서 선택한 게시된 글의 상세 내용을 확인하는 페이지다.

[게시글 상세 페이지 - 게시글 정보]
- 글 본문 상단에서 다음 정보를 확인할 수 있다.
    - 글의 유형, 제목, 작성자, 작성자 등급, 작성일, 조회수, 좋아요 수

[게시글 상세 페이지 - 댓글 기능]
- "게시글 상세 페이지" 하단에서 댓글을 확인하고 작성할 수 있다.
- 로그인 상태:
    - 댓글 입력 후 댓글 작성 버튼으로 댓글을 등록할 수 있다.
- 미로그인 상태:
    - 댓글 입력은 불가하며 로그인 후 댓글 작성 안내가 표시된다.

[게시글 상세 페이지 - 수정 및 삭제]
-로그인 상태이며 작성자인 경우:
    -"게시글 상세 페이지" 우측 상단에서 수정 및 삭제 버튼을 사용할 수 있다.

[게시글 상세 페이지 - 좋아요 기능]
- 로그인 상태: 
    - 좋아요 버튼을 누르면 게시글을 추천할 수 있다. 좋아요 수가 증가한다.
    - 다시 클릭하면 좋아요 취소가 된다.
- 미로그인 상태:
    - 로그인 후 좋아요 버튼으로 표시되며 기능은 비활성화 된다.
    
[계정 찾기 페이지 - 개요]
- 가입 시 등록한 정보로 계정(이메일)을 찾을 수 있는 페이지다.

[계정 찾기 페이지 - 계정 조회]
- 이용자의 이름과 생년월일을 입력할 수 있다.
- 다음 버튼 클릭 시:
    - 가입 시 설정한 계정 찾기 질문이 표시된다.
- 질문에 대한 답변을 작성 후 계정 확인을 누르면 가입된 계정(이메일)을 알려준다.
- 다시 입력하기 버튼을 눌러 이름과 생년월일을 다시 입력할 수 있다. 

[계정 찾기 페이지 - 계정 확인 후 동작]
- 계정확인 후 다음 버튼을 제공한다.
    - 로그인하러 가기: "로그인 페이지"로 이동한다.
    - 다시 찾기: 이름과 생년월일 입력 단계로 돌아간다.  

[공고 찾기 페이지 - 개요]
- 사이트에 등록되어 있는 공고들을 확인할 수 있는 페이지다.

[공고 찾기 페이지 - 공고 목록]
- 각 공고에 대해 다음 정보를 확인할 수 있다.
    -마감일
    -공고명
    -발주기간
    -예산
    -상태에 
- 공고 클릭 또는 상세 버튼 클릭 시 "공고 상세 페이지"로 이동한다.

[공고 찾기 페이지 - 장바구니]
- 각 공고의 담기 버튼 클릭 시 해당 공고를 장바구니에 등록할 수 있다.

[공고 찾기 페이지 - 검색 및 필터, 정렬]
- 키워드 입력을 통해 공고를 필터링할 수 있다.
    - 공고명/기관/예산/마감 기준
- 필터 초기화 버튼 오른쪽에 다음 필터 및 정렬을 사용할 수 있다. 
    - 기관명 필터
    - 마감일, 이름 순 정렬
    - 표시 공고 개수

[공고 찾기 페이지 - 기타 기능]
- 새로고침 버튼을 클릭하면 공고 목록을 최신화한다.
- 필터 초기화 버튼을 클릭 시 입력한 필터와 정렬을 초기화 한다. 
- 페이지 하단의 페이지네이션을 통해 목록 페이지를 이동할 수 있다.
    - Previous / Next / 페이지 번호 직접 선택
    
[공고 상세 페이지 - 개요]
- 선택한 공고의 상세 정보를 확인하는 페이지다.

[공고 상세 페이지 - 상단 버튼]
- 페이지 상단에는 다음 버튼이 있다.
    - 뒤로가기
    - 목록으로
    - 장바구니 담기
-뒤로가기 클릭시 이전 페이지로 이동한다.
-목록으로 클릭시 "공고찾기 페이지"로 이동한다.
- 장바구니 담기 클릭 시 버튼 캡션은 장바구니 담김으로 변경되고 해당 공고가 장바구니에 추가된다.

[공고 상세 페이지 - 기본 정보]
- 다음 정보를 표시한다.
    - 공고 종류
    - 공고 진행여부
    - 마감임박 여부
    - 공고 제목
    - 공고번호
    - 발주기관
    - 지역
    - 예산
    - 공고게시일
    - 마감일
- 첨부파일 다운로드 기능을 제공한다.

[공고 상세 페이지 - 기능]
- "공고 상세 페이지" 중앙에 다음 버튼이 있다.
    - AI 요약
    - 서류 체크리스트
    - 투찰가이드

[공고 상세 페이지 - AI 요약]    
- AI가 공고문을 분석하여 요약 정보를 제공한다.
    - 공고 요약
    - 참가 자격
    - 실적 요건
    - 제출서류
    - 낙찰가 예측
    - 권고 액션
- AI 분석 리포트 다운로드 버튼을 통해 리포트를 다운로드 할 수 있다.

[공고 상세 페이지 - 서류 체크리스트]
- 제출해야할 서류들을 체크리스트 형태로 제공한다.
- 항목 클릭시 체크가 되어 진행률이 반영 된다.
- 다시 클릭시 체크가 해제된다.

[공고 상세 페이지- 투찰 가이드]
- AI 예측 기반 투찰 범위를 제공한다.
- 권고 액션을 함께 제시한다.

[공지사항 페이지 - 개요]
- 공지 사항을 확인할 수 있는 페이지다.
    - 공지사항 제목
    - 공지 유형
    - 작성일
    - 작성자

[공지사항 페이지 - 검색 및 필터]
- 검색창을 통해 공지사항을 검색할 수 있다.
- 유형별 필터 버튼을 제공한다.
    - 서비스 / 업데이트 / 점검 / 정책

[공지사항 페이지 - 상세 이동]
- 각 공지사항의 상세 버튼을 클릭 시 "상세 페이지"로 이동한다.
 
[대시보드 페이지 - 상단 지표]
- 다음 정보를 숫자로 제공한다.
    - 오늘 시작한 신규 공고 수
    - 관심 공고 수
    - 마감 3일 이내 공고 수
    - 관심 공고 예산 총합

    -신규 공고를 클릭하면  오늘 시작한 신규 공고를 보여준다.
    -관심 공고를 클릭하면 "장바구니 페이지"로 이동한다.
    -마감 임박을 클릭하면 3일 이내 마감 임박 공고를 보여준다.

[대시보드 페이지 - 차트]
- 월별 공고 추이를 선 그래프로 표시한다.
    - 기준 월 변경 가능
    - 마감일 기준
    - 최대 4개월 이후까지 조회
- 지역별 공고 분포를 막대 그래프로 표시한다.

[대시보드 페이지 - 맞춤 추천]
- 장바구니에 공고가 있는 경우 맞춤 추천 공고를 표시한다.
- 각 공고에 대해 다음 버튼을 제공한다.
    -담기
    -상세보기
- 담기를 클릭해 장바구니에 담을 수 있다.
- 상세보기를 통해 "공고 상세 페이지"로 이동한다.

[로그인 페이지 - 개요]
- 웹사이트에 로그인을 할 수 있는 페이지이다.
- 비로그인 상태에서 로그인 필수 기능에 접근할 경우 "로그인 페이지"가 노출된다.

[로그인 페이지 - 로그인 기능]
- 이용자는 이메일과 비밀번호를 입력하여 로그인할 수 있다.
- 로그인 성공 시 이전에 접근하려던 페이지 또는 "메인 페이지"로 이동한다.

[로그인 페이지 - 하단 이동 버튼]
- 하단에 다음 버튼이 제공된다.
    - 회원가입: "회원가입 페이지"로 이동한다.
    - 계정 찾기: "계정 찾기 페이지"로 이동한다.
    - 비밀번호 찾기: "비밀번호 찾기 페이지"로 이동한다.

[마이페이지 - 개요]
- 로그인한 사용자의 계정 정보와 개인 설정을 관리하는 페이지다.

[마이페이지 - 상단 정보]
- "마이 페이지" 상단에서 다음 정보를 확인할 수 있다.
    - 이용자 이름
    - 이용자 이메일
    - 계정 상태

[마이페이지 - 정보 탭 전환]
- 다음 버튼을 클릭해 표시되는 정보를 전환할 수 있음
    - 계정 정보
    - 회사 정보
    - 관심 키워드
    - 최근 본 공고

[마이페이지 - 계정 정보]
- 이용자 계정의 이름, 이메일 정보를 확인할 수 있다.
- 비밀번호 변경 기능을 제공한다.
    - 현재 비밀번호 입력
    - 새 비밀번호 입력
    - 새 비밀번호 확인 입력
    - 모두 입력 후 변경사항 저장 클릭 시 비밀번호가 변경된다.

[마이페이지 - 화면 설정]
- 계정 정보 하단에 설정 영역이 있다. 
- 야간 모드(다크 모드) 스위치를 통해 테마를 변경할 수 있다.
    - 스위치 ON: 다크 모드 적용
    - 스위치 OFF: 기본 테마 적용

[마이페이지 - 위험 영역]
- 계정 정보 최하단에 위험 영역이 있다.
- 회원 탈퇴 버튼을 통해 회원탈퇴를 할 수 있다.

[마이페이지 - 회사 정보]
- 회사명과 직위를 입력해 저장할 수 있다.
- 이미 저장된 정보가 있는 경우:
    - 새로운 회사명과 직위를 입력해 정보 업데이트가 가능하다.

[마이페이지 - 괌심 키워드]
- 다음 정보를 설정해 관심 키워드를 추가할 수 있다.
    - 키워드
    - 기초금액 최소값
    - 기초금액 최대값
- 각 키워드 우측의 쓰레기통 버튼 클릭 시 키워드를 삭제할 수 있다.

[마이페이지 - 최근 본 공고]
- 사용자가 최근 조회한 입찰 공고 목록을 확인할 수 있다.

[메인페이지 - 개요]
- 공고 검색, 주요 기능 빠른 이동, 추천 공고 확인, 로그인 상태 확인을 제공하는 페이지다.

[메인페이지 - 공고 검색]
- "메인 페이지" 상단에서 공고를 키워드로 검색 할 수 있다.
    - 공고명
    - 기관
    - 예산
    - 마감

[메인 페이지 - 공고 검색]
- "메인 페이지" 상단에서 공고를 키워드로 검색 할 수 있다.
    - 공고명
    - 기관
    - 예산
    - 마감일

[메인 페이지 - 바로가기]
- "메인 페이지" 중앙에 바로가기 영역이 있다.
- 다음 페이지로 빠르게 이동할 수 있다.
    - "대시보드 페이지"
    - "공고 찾기 페이지"
    - "장바구니 페이지"
    - "커뮤니티 페이지"

[메인 페이지 - 장바구니 / 대시보드 접근]
- 장바구니 영역에서 장바구니에 담긴 공고 수를 확인할 수 있다.
- 장바구니 클릭 시:
    - 로그인 상태: "장바구니 페이지"로 이동한다.
    - 미로그인 상태: "로그인 페이지"로 이동한다.

- 대시보드 클릭 시:
    - 로그인 상태: "대시보드 페이지"로 이동한다.
    - 미로그인 상태: "로그인 페이지"로 이동한다.

[메인 페이지 - 오늘의 추천]
- "메인 페이지" 하단에 오늘의 추천 영역이 있다.
- 다음 정보를 확인할 수 있다.
    - 오늘 등록된 신규 공고 수
    - 마감까지 3일 이내 남은 공고 수

[메인 페이지 - 오늘의 추천 이동]
- 신규 공고 클릭 시:
    - 로그인 상태: "대시보드 페이지"로 이동하여 신규 공고를 표시한다.
    - 미로그인 상태: "로그인 페이지"로 이동한다.
- 마감 임박 클릭 시:
    -로그인 상태: "대시보드 페이지"로 이동하여 마감 임박 공고를 표시한다.
    - 미로그인 상태: "로그인 페이지"로 이동한다.

[메인 페이지 - 로그인 상태 영역]
- "메인 페이지" 우측에서 로그인 상태를 확인할 수 있다.
- 미로그인 상태:
    - 이메일 / 비밀번호 입력 후 로그인 가능
    - "회원가입 페이지" / "계정 찾기 페이지" / "비밀번호 찾기 페이지"로 이동 가능
- 로그인 상태:
    - 프로필 수정 버튼 클릭 시 "마이페이지"로 이동한다.
    - 로그아웃 버튼 클릭 시 로그아웃 처리된다.

[비밀번호 찾기 페이지 - 개요]
- 임시 비밀번호 발급 전 본인 확인을 수행하는 페이지다.

[비밀 번호 찾기 페이지 - 본인 확인]
- 다음 정보를 입력한다.
    - 이름
    - 이메일
    - 생년월일
- "다음" 버튼 클릭 시 가입 시 설정한 질문이 표시된다.
- 질문에 대한 답변이 일치하면 맞는 본인 인증 단계가 완료된다. 

[비밀번호 찾기 페이지 - 임시 비밀번호 발급]
- "임시 비밀번호 전송" 클릭 시 임시 비밀번호가 이메일로 전송된다.
- "다시 입력하기 클릭 시 본인 확인 단계로 돌아간다.
- 버튼을 눌러 "로그인 페이지"로 이동 할 수 있다.
- 계정 찾기 버튼을 눌러 "계정 찾기 페이지"로 이동할 수 있다.

[새 글 작성 페이지 - 개요]
- 커뮤니티에 새로운 게시글을 작성할 수 있는 페이지다.

[새 글 작성 페이지 - 작성 기능]
- 게시글 유형을 카테고리 버튼으로 선택할 수 있다.
- 제목과 본문 내용을 직접 입력한다. 
- 첨부파일 업로드 기능을 제공한다.

[새 글 작성 페이지 - 등록 및 취소]
- 등록 클릭 시 게시글이 커뮤니티에 게시된다.
- 취소 클릭 시 "커뮤니티 페이지"로 이동한다.

[알림 페이지 - 개요]
- 관심 공고 및 키워드 기반 알림을 확인하고 설정하는 페이지다.

[알림 페이지 - 알림 유형]
- 다음 알림 유형을 제공한다.
    - 마감 임박
    - 마감됨
    - 키워드 알림
- NEW 표시가 있는 알림은 클릭 시 표시가 해제된다.

[알림 페이지 - 필터 탭]
- 알림 센터 하단에서 다음 탭을 선택할 수 있다.
    - 전체
    - 긴급
    - 키워드
    - 설정

[알림 페이지 - 탭별 동작]
- 전체: 모든 알림을 표시한다.
- 긴급: 마감 입박, 마감됨 알림만 표시한다.
- 키워드: 설정한 키워드를 포함한 공고 알림을 표시한다.
- 설정:
    - 관심 키워드 등록 및 삭제
    - 기초금액 최소/최대 설정
    - 마감 임박/마감 알림 수신 여부 설정
    - 정정/재공고 알림 수신 여부 설정

[알림 페이지 - 기타 기능]
- 새로고침 버튼 클릭 시 새로운 알림을 조회한다.
- 모두 읽음 클릭 시 모든 알림을 읽음 처리한다.

[장바구니 페이지 - 개요]
- 사용자가 관심 공고로 등록한 공고를 관리하는 페이지다.
- 공고를 직접 클릭하여 해당 공고의 "공고 상세 페이지"를 확인할 수 있다.
- 각 공고 우측의 쓰레기통 모양 버튼을 클릭하면 관심 공고를 장바구니에서 삭제한다.

[장바구니 페이지 - 상태별 필터]
- 다음 상태별 관심 공고 개수를 표시한다.
    - 관심
    - 검토중
    - 참여결정
    - 서류준비
    - 제출완료
- 상태 클릭 시 해당 상태 공고만 필터링된다.
- 해제 클릭 또는 재클릭시 필터가 해제된다.

[장바구니 페이지 - 관심 공고 목록]
- 마감되지 않은 관심 공고에 대해 다음 정보를 표시한다.
    - 공고명
    - 마감일까지 남은 기간
    - 발주기관
    - 예산
    - 마감일
- 상태 변경 드롭다운을 통해 공고 상태를 변경할 수 있다.
- 정렬 옵션을 제공한다.
    - 마감 빠른순 / 마감 늦은순 / 제목순

[장바구니 페이지 - 지난 공고 목록]
- 마감된 공고에 대해 다음 정보를 표시한다.
    - 공고명
    - 투찰 여부
    - 낙찰 여부
    - 발주기관
    - 예산
    - 마감일
- 상태를 미투찰 / 낙찰 / 탈락으로 변경할 수 있다.

[챗봇 모달 - 개요]
- 공고 검색, 사이트 이용 안내, 공고문 요약을 제공하는 챗봇 모달이다.

[챗봇 모달 - 제공 기능]
- 사이트 이용 방법에 대한 질문을 할 수 있다.
- 조건(공고명, 공고번호, 공사지역, 기관명, 공고시작/마감/개찰일, 기초금액, 낙찰하한율, 예가범위 상하한) 기반 공고 조회 질문을 할 수 있다.
- 공고문 파일을 업로드해 요약 리포트를 생성할 수 있다.

[챗봇 모달 - 제한 사항]
- 사이트 이용 및 공고 조회 질문 외 질문에는 답변하지 않는다.

[챗봇 모달 - 기타 기능]
- 전송 버튼 클릭 시 질문에 대한 답변을 생성한다.
- 빠른 질문을 통해 미리 정의된 질문을 선택할 수 있다.
- 클립 버튼 클릭 시 파일을 업로드할 수 있다.

[회원가입 페이지 - 개요]
- 신규 사용자가 회원가입을 진행하는 페이지다.

[회원가입 페이지 - 입력 정보]
- 다음 정보를 입력한다.
    - 이름
    - 이메일
    - 닉네임
    - 생년월일

[회원가입 페이지 - 비밀번호 설정]
- 비밀번호는 비밀번호 규칙 체크리스트를 만족해야한다.
- 체크리스트를 하나라도 만족하지 않으면 가입하기 버튼은 비활성화된다.
- 비밀번호 규칙 체크리스트에서 하나라도 만족하지 않을 시 가입하기 버튼은 활성화 되지 않는다.
- 비밀번호 확인은 동일한 값으로 입력해야 한다.

[회원가입 페이지 - 계정 찾기 정보]
- 계정 찾기 질문을 선택한다.
- 계정 찾기 답변을 입력한다.

[회원 가입 페이지 - 약관 동의]
- 필수 약관 항목은 반드시 동의해야 한다.

[회원가입 페이지 - 가입 완료]
- 모든 정보 입력 완료 시 가입하기 버튼이 활성화된다.
- 가입하기 클릭 시 회원가입이 완료된다.

[추천 공고 팝업 - 개요]
- 사용자 관심 조건과 활동 이력을 기반으로 공고를 추천하는 팝업이다.

[추천 공고 팝업 - 노출 조건]
- 로그인 상태에서 "공고 찾기 페이지" 진입 시 팝업이 노출된다.

[추천 공고 팝업 - 기능]
- 각 공고에 대해 다음 버튼을 제공한다.
    - 상세 보기
    - 장바구니 담기

[추천 공고 팝업 - 닫기]
- 우측 상단 X 버튼 또는 하단 닫기 버튼 클릭시 팝업이 닫힌다.
- 오늘 다시 보지 않기 선택 시 하루 동안 팝업이 노출되지 않는다.
